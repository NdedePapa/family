🔄 Clear Cache & Fix Errors

If you're seeing errors like:

❌ Failed to load statistics
Unexpected token '<'

Solution:

Your browser is serving old cached files. Click the button below to clear everything:

Manual Method (if button doesn't work):

  1. Press Ctrl + Shift + Delete
  2. Select "Cached images and files"
  3. Click "Clear data"
  4. Close this tab
  5. Reopen http://localhost:3000

Or use DevTools:

  1. Press F12
  2. Right-click the refresh button (⟳)
  3. Select "Empty Cache and Hard Reload"
